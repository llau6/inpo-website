Program Analyst

Co-op students in this role perform a variety of technical support duties including, but not limited to the following: phone support and computer support for issues and/or questions being reported to the Help Desk. The ideal candidate would be one who excels in a highly stressful environment, working within tight deadlines and demanding users, and doing so with impeccable customer service skills. The candidate will be required to work effectively within a tight-knit team structure.

Essential Duties


Develop Windows client and web-based software using .NET 4.0 Framework, Visual Studio 2010, and Team Foundation Server. Enhance and maintain existing Active Server Pages applications that use DHTML and XML on our intranet and external web site.

Program SQL data retrieval language from Microsoft SQL Server 2008 databases on the local area network. Develop Web software using IIS (Microsoft's Internet Information Server).

Perform batch programming on plant information center (PIC) data gathering and processing software, program in macro languages (VBA). Troubleshoot application code and data-related problems with production applications received from the IT Help Desk. Perform systems analysis and user interface software screen design. Process and filter descriptions of events at plants, including data entry and collating of information received on plants. Perform programming and analysis tasks on IT work requests as assigned by project leaders. Generate graphics, charts, displays with data received on nuclear plant operations

Develop software to process data and interface with Microsoft Excel to generate trend graphs and other graphic outputs.
